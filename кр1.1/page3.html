Задача 3: Решение алгоритмической задачи

Задача: Найти максимальную сумму непрерывного подмассива (Алгоритм Кадане)

Описание: Дан массив целых чисел, найти непрерывный подмассив с наибольшей суммой.

Пример: Массив [-2, 1, -3, 4, -1, 2, 1, -5, 4] → Максимальная сумма: 6

Реализация на JavaScript:

function maxSubArray(nums) {
    let maxSum = nums[0];
    let currentSum = nums[0];
    
    for (let i = 1; i < nums.length; i++) {
        currentSum = Math.max(nums[i], currentSum + nums[i]);
        maxSum = Math.max(maxSum, currentSum);
    }
    
    return maxSum;
}
